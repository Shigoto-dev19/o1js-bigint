# Mina zkApp: Provable Bigint

This template uses TypeScript.

## How to build

```sh
npm run build
```

## How run and plot benchmarks

```sh
node build/src/benchmark_runner.js
```

```sh
python3 src/plot.py
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
